# find.bi Frontend

React 18 + Vite + TypeScript frontend for find.bi.

## Tech Stack

- React 18 + TypeScript
- Vite (build tool)
- Tailwind CSS (styling)
- Apache ECharts (dashboard charts)
- Observable Plot (AI exploratory charts)
- Deck.gl + MapLibre GL JS (geo-visualization)
- DuckDB-WASM (client-side SQL engine)
- Monaco Editor (SQL editor)
- react-grid-layout (dashboard layout)

## Setup

```bash
npm install
npm run dev
```

## Scripts

```bash
npm run dev        # Start dev server
npm run build      # Production build
npm run typecheck  # TypeScript check
npm run lint       # ESLint
npm test           # Vitest
```
